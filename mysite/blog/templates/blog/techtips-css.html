HTML Elements

<html>

  • usually found at the top of the file and has closing tag at the end.
  • Describes a file type to be HTML
  • The start tag may be omitted if the first thing inside the <html> element is not a comment.
  • The end tag may be omitted if the <html> element is not immediately followed by a comment.
  • Used in standard HTML practices
  • To learn more, Click the tag above

<h1> .... <h6>

  • Header Tags make different sizes of headers for titles and headers on a web page
  • usually found at the top of the file and has closing tag at the end.
  • 6 different sizes used as tags h1-h6 with h1 being the biggest
  • Header tags are inline level elements
  • To learn more, Click the tag above

<p>

  • The <p> HTML element represents a paragrap
  • It is a block level element
  • Can be used with margins and padding to create areas/blocks of content.
  • To learn more, Click the tag above

<title>

  • The <title> HTML element is used in the head of an html file and defines the document's title that is displayed in the address bar.
  • It is neither an inline nor a block level element
  • Only contains text inside the tags, all other tags will be ignored
  • To learn more, Click the tag above

HTML HTML Character Entities

& l t ; = < Adding these characthers without a space makes the less than sign

  • Used for math expressions
  • Use in tags when writting HTML, but when attempting to put the < character in text, there is an error in the code for an incomplete tag. So this character entity is useful.

& g t ; = > Adding these characthers without a space makes the greater than sign

  • Used for math expressions
  • Use in tags when writting HTML, but when attempting to put the > character, there is an error in the code for an incomplete tag. So this character entity is useful.

& quot ; = " Adding these characthers without a space maske a double quotation marks symbol

  • Used when quoting somone.
  • Use when declaring an attribute when writting HTML, but when attempting to put the < character in text, there is an error in the code for an incomplete attribute assignemt. So this character entity is useful.

CSS Selectors

element

  • The element selector allows you to define CSS declarations that happen every time a certin element appears in a html document.
  • For example, if you wanted every <p> to be blue, you could make a selector for <p> with color: blue; in it.

.class

  • The .class selector allows you to describe how any element should look in an html document. This means you can change the appearnce, from the already established declarations by adding class= "classname" in the tag.
  • You can change any stye elements in a .class selector

#id

  • The #id element is like the .class element. It allows you to specifiy CSS declarations for any tag that has the id referenced in it. However, #id's can also be used inside classes.
  • The #id is declared by using # following the name of the id, for example #intro
  • The #id is declared in a html tag with id="tagname"

:nth-child

  • The :nth-child(n) selector matches every element that is the nth child, regardless of type, of its parent.
  • The n can be a number, keyword, or a formula
  • This allows children elements to look or be defined different from each other.
  • ex. p:nth-child(2) { background: red; } will make every element that is the 2nd child of <p> red.

CSS Properties

background-color

  • background-color allows you to change what color the background behind the text.
  • The background-color will change the background of the entire space that the text takes up, so for block elements, it will be the whole block, and for inline it will change the line.
  • There are many built in colors, like red, orangred, lightgreen, and almost anything you can think of.
  • There is also the option to choose your own combination of rgb() to make any color available.

font-family

  • font-family is used with the style element and allows you to chooes what font the text looks like.
  • Some font families are singular, like monospace and cursive
  • Others contain a family of fonts, like: Cambria, Cochin, Georgia, Times, 'Times New Roman', serif

font-size

  • font-size changes the size of the font of a text
  • You can change the font size to be bigger or smaller and just put in the number with the units after it, for example, font-size: 35pt;
  • It is recommended to use headers instead of changing the font size for titles and sections of text.

margin

  • margin sets the margin for all four sides of an element
  • ex. margin: 35px (set it to 35 pixles)
  • There are also the options to set only one side of an element, by using the property margin-top, margin-right, margin-left, or margin-bottom.
